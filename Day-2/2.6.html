Tag Description
<html>--</html> Declares the webpage to be written in HTML.
<head>--</head> Delimits the page's head
<title>--</title> Defines the title of the page (not displayed on the page.)
<body>--</body> Delimits the page's body.
<hr>--</hr> Delimits a level n heading.
<b>--</b> Set text in bold face.
<i>--</i> Set text in italic.
<center>--</center> Keeps elements center on the webpage horizontally.
<ul>--</ul> Helps in creating ordered list.
<ol>--</ol> Helps while creating un-ordered list.
<li>--</li> Used to declare list items on the webpage.
